# 10817. 세 수

## 문제

세 정수 A, B, C가 주어진다. 이때, 두 번째로 큰 정수를 출력하는 프로그램을 작성하시오. 



### 입력

첫째 줄에 세 정수 A, B, C가 공백으로 구분되어 주어진다. (1 ≤ A, B, C ≤ 100)

### 출력

두 번째로 큰 정수를 출력한다.



## 예제

### 입력1

```python
20 30 10
```

### 출력1

```python
20
```



### 입력2

```python
30 30 10
```

### 출력2

```python
30
```



### 입력3

```python
40 40 40
```

### 출력3

```python
40
```



### 입력4

```python
20 10 10
```

### 출력4

```python
10
```





## 코드

```python
arr = list(map(int, input().split()))
print(sorted(arr)[1])
```
